# Health Care App

## Overview
This is a telemedicine Android application developed as part of the CSE-2216: Application Development Lab project. The app facilitates healthcare services, allowing doctors and patients to manage profiles, upload images, and interact through Firebase-backed features.

## Features
- User profiles for doctors and patients
- Image upload functionality for profile pictures
- Firebase integration for data storage and authentication
- Profile editing capabilities
- Unit testing with JUnit

## Technologies Used
- **Language**: Java
- **Platform**: Android
- **Build System**: Gradle
- **Backend**: Firebase (Realtime Database, Storage)
- **Dependencies**: AndroidX, Google Services
